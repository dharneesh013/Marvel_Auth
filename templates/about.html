Marvel - Authentication


Marvel-Authentication was a Graphical Password Authentication platform
designed  to  overcome difficulties  raised  from the normal  text  -  based
Password  Authentication  system. Register   an  account  by  entering  the
username,  email  and  select  the  grids  for  the  password.  Then login to
see the change in Marvel-Auth status. Remember the email registered with
the  password  to  receive  the  notification  mail   on  reset  password page
